## Markdown

Markdown es un lenguaje de marcas pensado para documentacion y para ser convertido a otros formatos.

### Cabeceras

Las cabeceras se marcan con tantas almohadillas como nivel de cabecera: # es h1; ## es h2; ... asi hasta h6.
